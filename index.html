Visual Studio Code

Visual Studio Code, commonly referred to as VS Code,[9] is an integrated development environment developed by Microsoft for Windows, Linux, macOS and web browsers.[10][11] Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded version control with Git. Users can change the theme, keyboard shortcuts, preferences, and install extensions that add functionality. Visual Studio Code is proprietary software released under the "Microsoft Software License",[6] but based on the MIT licensed program named "Visual Studio Code — Open Source" (also known as "Code — OSS"), also created by Microsoft and available through GitHub.[12] Open-source binaries are available from VSCodium, without Microsoft's telemetry. In the Stack Overflow 2023 Developer Survey, Visual Studio Code was ranked the most popular developer environment tool among 86,544 respondents, with 73.71% reporting that they use it.[13]

features of vscode

Visual Studio Code (VS Code) is a source code editor with many features, including:



Editing: Features include syntax highlighting, bracket-matching, auto-indentation, and multiple cursors for simultaneous editing
Developer tooling: Includes IntelliSense code completion, debugging, semantic code understanding, and navigation
Version control: Built-in support for Git and other source control providers
Testing: Run automated tests and view test coverage
Customization: Users can change the theme, keyboard shortcuts, preferences, and install extensions
Split view: Allows users to work on two different files simultaneously or check the difference between two files
Navigation bar: Also called breadcrumbs, this feature shows the file path and symbol path up to the cursor position
Plugins: Users can create their own plugins, such as Settings Sync to sync VS Code installations on different devices
Integrated command-line terminal: Users can open files, install extensions, change the display languages, and output diagnostics

: Adds attributes to an element

!: Adds a comment to an element

SHORTCUT KEYS

Keyboard shortcut keys are special keys on a keyboard that allow users to perform tasks more quickly.

keys DESCRIPTION
Ctrl + S To save the current file
Ctrl + O To open the selected file
Ctrl + C To copy the selected text
Ctrl + v To paste the selected text
Ctrl + x To cut the text
Ctrl + z To return the previous text
Ctrl + A To select all content file
Ctrl + Shift + p To open command palette
Ctrl + Shift + N To open new window
